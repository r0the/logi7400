# 7404: six hex inverters

- Type: [gate](gates.md)
- DIP: 14-pin
- Number of elements: 6
- Inputs per element: 1
- Outputs per element: 1

## Description

Provides six hex inverters.

## Inputs and outputs

| Label | Description            |
| ----- | ---------------------- |
| An    | Input of inverter n    |
| Yn    | Output of inverter n   |

## Function table

| An  | Yn  |
|:---:|:---:|
| L   | H   |
| H   | L   |

- H: HIGH voltage level
- L: LOW voltage level
