# 7442: BCD to decimal decoder, active low output

- Type: [decoder](encoders_decoders.md)
- DIP: 16-pin
- Input: 4-bit BCD
- Output: 10 lines

## Description

Decodes a 4-bit BCD to ten output lines.

## Inputs and outputs

| Label | Description  | Signal      |
| ----- | ------------ | ----------- |
| Dn    | input lines  | active high |
| Yn    | output lines | active low  |
